    Samples by Class: ms-bgColor-*¶

    The ms-bgColor-* classes apply a background color. Several standard, shared, theme, neutral, and more are included.

    Samples are grouped by classes used. Expand to see the related samples.

    Complimentary Classes

    These classes compliment the ms-borderColor-* and ms-fontColor-* classes and except where noted below contain equivalent colors.

    Warning

    The named color alert does not have a ms-bgColor- equivalent despite being present elsewhere
